Определитель матрицы:

    gcc maindet.c testdet.c matrix.a -o dettest.out

    1-ый тест: 5 заданных матриц из папки Determ.
    2-ой тест: Набор сатриц со случайным наполнением и размерами, составлющую геометрическую прогрессию (при n > 64 происходит переполнение)

Ранг матрицы:

    gcc mainRank.c testRank.c matrix.a -o ranktest.out

    1-ый тест: 5 заданных матриц из папки Rank.
    2-ой тест: В матрице разера 5 на 5, заполненной единицами элементы случайным образом меняются на о


Решение СЛУ:

    gcc mainSLE.c testSLE.c matrix.a -o SLEtest.out

    1-ый тест: Создает матрицы размера 6 12^n, и соответсвующие столбцы свободных членов так, 
    что СЛУ имеет повно одно решение (1, 1 ,1, ...). После считает время выполнение и 
    среднеквадратичное отклонение значений.

Обратная матрица:

    gcc maininv.c testinv.c matrix.a -o invtest.out

    1-ый тест: 5 заданных матриц из папки inc.
